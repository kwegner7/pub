This is a coding example of a Jupyter Notebook, coded in Python. This Jupyter Notebook captures a snapshot, in real-time for the current day, of objects in earth orbit. The user can select for display: GPS Satellites, Space Stations, International Space Station and/or Chinese Anti-Satellite Missile Test Debris. From the link, Open with Google Colaboratory, pulldown “Runtime” and click on “Run all”. At the bottom is an interactive display of these space objects. Click anywhere on the plot area at the bottom and scroll up to see the controls.
