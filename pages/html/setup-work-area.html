SET UP THE WORK AREA

The Radiation Effects Software can be run inside or outside of the classified area. Inside of ADTEC, the classified database is used. Outside of ADTEC, the unclassified database is used.

There is a set of historical RES version packages available at the Navy SP unclassified server. Each package is complete and sufficient to compile, run and regression test the program. The name of the folder is the date of the version:

\\usqts1nas01\COS1\USCOS1\NavySP\Repo\RES\versions\2020-04-06
\\usqts1nas01\COS1\USCOS1\NavySP\Repo\RES\versions\2020-04-17
. . .
. . .

UNCLASSIFIED WORK AREA

Copy the version package to a local work area on your unclassified office PC. For example, create a folder at k:\work to contain the RES package. The unclassified database will be located at:

k:\work\tools\genfit.db3

CLASSIFIED WORK AREA

Create a DVD from the version package. Schedule a DTA to move the DVD into ADTEC. Copy the version package to your user space on a Windows 10 computer in the ADTEC. For example, create a folder at k:\work to contain the RES package. The unclassified database will be located at:

k:\work\tools\genfit.db3

Replace the unclassified version of the database with the classified version by copying genfit.db3 from:

\\navyfs04\GIT_Repository\RadiationEffectsSoftware

VERSION PACKAGE

Open the PowerShell and type:

cd k:\work
 dir

There are five folders in the package:

notes
The notes folder contains instructions for compiling, running and performing regression tests.

src
The src folder contains the source code for Radiation Effects Software.

run
The run folder contains the input and output xml and csv files.

tools
The tools folder contains scripts for compiling, running and performing regression tests. This folder also contains the executable file, the dll files and the database file.

xerces
The xerces folder contains the third party software used for reading and writing xml files.
