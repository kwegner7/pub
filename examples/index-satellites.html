EXAMPLE PYTHON JUPYTER NOTEBOOK

This Jupyter Notebook captures a snapshot in real-time of objects in earth orbit. It downloads daily orbital object data from NORAD websites. The code is written in Python using Pandas dataframes and matplotlib to display the plots. The user can select for display:

  • GPS satellites
  • Space Stations
  • Chinese Anti-Satellite Missile Test Space Debris
  • The International Space Station

From the link, "Open with Google Colaborator", then pulldown “Runtime” and click on “Run all”. The code for the Python classes is visible to show the design. At the bottom of the notebook is an interactive display of these space objects.
